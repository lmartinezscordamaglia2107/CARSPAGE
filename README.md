# CarsPage 🚗✨

> Uma plataforma moderna, fluida e responsiva para exibição, busca e catalogação de veículos.

O **CarsPage** é um projeto desenvolvido para proporcionar uma experiência de usuário incrível na busca pelo carro dos sonhos. Com um design limpo, focado na usabilidade (UI/UX) e alta performance, a aplicação permite navegar por categorias, filtrar especificações e visualizar detalhes de diversos modelos de forma intuitiva.

---

## 🚀 Funcionalidades

- **Vitrine Dinâmica:** Exibição de veículos com imagens de alta qualidade e especificações rápidas.
- **Filtros Avançados:** Busca simplificada por marca, ano, faixa de preço e tipo de combustível.
- **Design Responsivo:** Experiência otimizada para dispositivos móveis, tablets e desktop.
- **Modo Escuro (Dark Mode):** Conforto visual para navegação noturna.
- **Página de Detalhes:** Visualização completa de ficha técnica, galeria de fotos e formulário de contato/proposta.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e ferramentas:

- **[React](https://reactjs.org/)** – Biblioteca Javascript para construção de interfaces.
- **[Tailwind CSS](https://tailwindcss.com/)** – Framework utilitário de CSS para estilização rápida e moderna.
- **[Vite](https://vitejs.dev/)** – Ferramenta de build extremamente rápida para o ecossistema frontend.
- **[Lucide React](https://lucide.dev/)** – Pacote de ícones limpos e modernos.
